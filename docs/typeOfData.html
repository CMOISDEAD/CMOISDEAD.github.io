# Name Description Value
1 byte Representa un tipo de dato de 8 bits con signo. De tal manera que puede almacenar los valores numéricos de -128 a 127 (ambos inclusive). 0
2 short Representa un tipo de dato de 16 bits con signo. De esta manera almacena valores numéricos de -32.768 a 32.767. 0
3 int Es un tipo de dato de 32 bits con signo para almacenar valores numéricos. Cuyo valor mínimo es -231 y el valor máximo 231-1. 0
4 float Es un tipo dato para almacenar números en coma flotante con precisión simple de 32 bits. 0
5 long Es un tipo de dato de 64 bits con signo que almacena valores numéricos entre -263 a 263-1 0
